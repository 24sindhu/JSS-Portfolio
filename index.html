Python, TensorFlow/Keras, OpenCV, CNN, Computer Vision

Deep learning system that classifies 156 handwritten Tamil script characters from grayscale images using a custom convolutional neural network and curated preprocessing pipeline.

  • Used the HPL Tamil Offline Character Dataset with 156 character classes and preprocessed grayscale TIFF images into centered 128×128 inputs.
  • Implemented bounding-box extraction, resizing to 100×100, padding to 128×128, and pixel normalization to standardize training data.
  • Designed a custom CNN with stacked Conv2D and MaxPooling2D layers, BatchNormalization, Dropout, and dense layers with ReLU and final softmax for 156-way classification.
  • Achieved around 97.8% train accuracy and 92.5% test accuracy on the base model, then experimented with L2 regularization and higher dropout for better generalization.
  • Provided a reusable prediction pipeline where new images are preprocessed and passed through the trained model to output the corresponding Tamil character code.
